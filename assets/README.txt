Положи сюда свои медиа-файлы:

  background.mp4   — видео-фон (1080p, без звука, 10–30 сек)
  poster.jpg       — постер для видео
  ambient.mp3      — фоновая музыка (loop)
  hover.mp3        — звук наведения на пункт меню (короткий)
  click.mp3        — звук клика/активации

Если файлов нет — сайт всё равно работает:
  • видео заменяется тёмным градиентом
  • звуки синтезируются через Web Audio API (неоновые бипы и ambient-pad)
