# ❀ vio — prologue

Сайт-визитка в стиле главного меню инди-игры. Вайб где-то между **NieR: Automata**, **dreamcore** и тёплым **moe**: пыльно-кремовый «бумажный» фон, чёрная италика с засечками, плавающие лепестки, мягкие колокольчики при наведении и тихая фоновая мелодия. Без сборщиков и зависимостей — просто открыть `index.html`.

## Что внутри

```
game-menu-site/
├── index.html      # разметка
├── styles.css      # сепия-палитра, орнаменты, плавные переходы
├── script.js       # бут-экран, навигация, лепестки, last.fm, звуки
├── config.js       # ★ личные данные: last.fm, ссылки, «обо мне»
├── README.md
└── assets/
    ├── background.mp4   # видео-фон  (опционально)
    ├── poster.jpg       # постер для видео (опционально)
    ├── ambient.mp3      # фоновая мелодия (опционально)
    ├── hover.mp3        # звук наведения (опционально)
    └── click.mp3        # звук клика (опционально)
```

## Заполни `config.js` под себя

Все «личные» данные собраны в одном файле — `config.js`. Структура:

```js
window.SITE_CONFIG = {
  about: {
    name:      "vio",
    realName:  "TODO: имя",
    location:  "Калининград, Россия",
    birthday:  "YYYY-MM-DD",   // возраст пересчитается сам
    intro:     "…",
    discordId: "YOUR_DISCORD_USER_ID",  // Lanyard
  },
  lastfm: { username, apiKey, … },
  links:  [ /* соц-сети */ ],
};
```

### Last.fm

1. Получить API-key: https://www.last.fm/api/account/create  
2. Вписать ник и ключ в `config.lastfm`.

⚠️ С 2022 last.fm API геоблокирован для РФ (отвечает 403). Локально для разработки — VPN. **На GitHub Pages / Netlify / Vercel запрос идёт с IP сервера и работает само.**

### Discord (через Lanyard)

Сайт показывает живой статус: аватар, ник, online/idle/dnd/offline, что играешь, что слушаешь в Spotify (если у тебя подключена интеграция Spotify ↔ Discord) и кастомный статус. Realtime через WebSocket — обновляется без перезагрузки.

Что нужно сделать (один раз):

1. Зайти на дискорд-сервер Lanyard: **https://discord.gg/lanyard**  
   Просто вступить, делать ничего больше не нужно — бот сам начнёт «видеть» твой статус.
2. В Discord: Settings → Advanced → включить **Developer Mode**.
3. Right-click по своему аватару → **Copy User ID**.
4. Вставить ID в `config.about.discordId`.

Если оставить плейсхолдер — Discord-блок просто не появится, всё остальное работает.

### Возраст

Считается автоматически из `config.about.birthday`. Формат — `YYYY-MM-DD`. Учитывает, был ли уже день рождения в этом году. Подпись склоняется по русской грамматике («24 года», «25 лет», «21 год»). Раз в час пересчитывается на случай, если страница открыта в полночь дня рождения.

### Ссылки

Массив `config.links`. Поля: `label`, `url`, `icon` (любой символ/эмодзи), опционально `note` — короткий хвостик-подпись.

- Клавиатура: `↑ ↓` / `W S` — листать, `Enter` / `Space` — открыть, `Esc` — назад
- Кнопка `мелодия · вкл/выкл` в правом нижнем углу — переключение звука

Внутри открытой страницы (например, «работы», «now playing») контент скроллится нативно колесом, если не помещается.


## Подмена медиа

Положи свои файлы в папку `assets/` с такими именами:

| Файл | Назначение | Что подходит к вайбу |
|---|---|---|
| `background.mp4` | Видео-фон | Тихие сцены: дождь по окну, ветви на ветру, поле травы, морская гладь, плёночные кадры. 1080p, без звука, 10–30 сек, < 8 МБ |
| `poster.jpg` | Кадр-плейсхолдер до загрузки видео | 1920×1080 |
| `ambient.mp3` | Фоновая мелодия (loop) | Фортепиано в духе Кэйити Окабэ (NieR), Joe Hisaishi, Ёко Симомура; тихий ambient/lo-fi/dream-folk. –12..–18 LUFS |
| `hover.mp3` | Очень тихий «дзинь» при наведении | < 100 мс, мягкий, чтобы не утомлять |
| `click.mp3` | Звон при выборе | Колокольчик/арфа, 200–600 мс |

Если каких-то файлов нет — сайт не сломается:
- **Видео** заменится мягким сепия-градиентом с двумя «ореолами» света (CSS-фоллбэк).
- **Звуки** автоматически синтезируются через Web Audio API: мягкие колокольчики для UI и тёплый ambient-pad с медленным фильтр-свипом для фона. Иногда поверх него тихо звенит случайная нота из пентатоники — словно мелодия в тумане.

Где взять бесплатные ассеты:
- Видео: [Pexels](https://www.pexels.com/videos/), [Coverr](https://coverr.co/), [Mixkit](https://mixkit.co/free-stock-video/)
- Звуки UI: [Freesound](https://freesound.org/), [Mixkit Sound Effects](https://mixkit.co/free-sound-effects/)
- Музыка: [Pixabay Music](https://pixabay.com/music/), [FreePD](https://freepd.com/), [Free Music Archive](https://freemusicarchive.org/)

Проверь лицензию на коммерческое использование, если нужно.

## Кастомизация

Все цвета и шрифты — CSS-переменные в начале `styles.css`:

```css
:root {
  --c-paper:      #efe6d3;   /* фон-«бумага» */
  --c-ink:        #1a1714;   /* чернила */
  --c-rose:       #f4b8c4;   /* пастельный розовый акцент */
  --c-rose-deep:  #d98aa0;
  --c-lilac:      #c9b9e0;
  --c-mint:       #b9d8c5;
  --c-gold:       #c79f5a;
  --c-blood:      #8b3a3a;   /* приглушённый бордо */
  --font-display: "Cormorant Garamond", ...;
  --font-body:    "EB Garamond", ...;
}
```

Контент меню и страниц-дневника — прямо в `index.html` (секции `<section class="panel" ...>`).

Хочется тёмный «ночной» вариант? Поменяй `--c-paper` на тёмно-чернильный (`#1a1714`), а `--c-ink` на кремовый (`#efe6d3`) — палитра инвертируется.

## Браузерная поддержка

- Современные Chromium / Firefox / Safari (актуальные версии).
- На iOS видео автоплей работает только когда `muted` + `playsinline` (уже выставлено).
- Автозапуск музыки везде блокируется до первого взаимодействия — поэтому фоновое стартует с первого клика/нажатия клавиши.
- Уважается `prefers-reduced-motion`: анимации почти отключаются, лепестки не появляются.
